# Qt HMI Test Automation Dashboard

A modern industrial-style dashboard built using **Qt Quick (QML)** for monitoring devices, running automated tests, and visualizing signals.

## Features

• Device monitoring panel  
• Signal analysis charts  
• Test automation control panel  
• Log monitoring system  
• Test result visualization  
• Modular QML architecture  

## Technologies

- Qt Quick / QML
- Qt Charts
- C++
- Qt Creator
